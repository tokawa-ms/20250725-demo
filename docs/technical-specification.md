# テトリスゲーム技術仕様書

## 概要

80年代レトロスタイルのテトリスゲームの技術仕様と実装詳細について記述します。

## システム要件

### 最小要件
- モダンWebブラウザ（Chrome 90+, Firefox 88+, Safari 14+）
- JavaScript ES6+ サポート
- Canvas API サポート
- Web Audio API サポート（音楽機能用）

### 推奨要件
- 1024x768 以上の画面解像度
- ステレオスピーカーまたはヘッドフォン
- キーボード（最適な操作体験用）

## ゲーム仕様

### ゲームボード
- サイズ: 10（幅）× 20（高さ）
- セルサイズ: 20px × 20px
- 描画方式: HTML5 Canvas

### テトロミノ
以下の7種類のテトロミノを実装：

| 種類 | 形状 | 色 |
|------|------|-----|
| I | 4×1直線 | シアン (#00ffff) |
| O | 2×2正方形 | イエロー (#ffff00) |
| T | T字型 | マゼンタ (#ff00ff) |
| S | S字型 | グリーン (#00ff00) |
| Z | Z字型 | レッド (#ff0000) |
| J | J字型 | ブルー (#0080ff) |
| L | L字型 | オレンジ (#ff8000) |

### ゲームルール
1. テトロミノは画面上部から出現
2. 重力により自動的に下降
3. プレイヤーは左右移動・回転・高速落下が可能
4. 横一列が埋まると消去される
5. 新しいテトロミノが出現できない場合はゲームオーバー

### 操作方法
| 入力 | 動作 |
|------|------|
| ← → | 左右移動 |
| ↑ | 回転 |
| ↓ | 高速落下 |
| SPACE | 即座に落下 |
| S | ゲーム開始 |
| P | 一時停止/再開 |
| R | リセット |

### スコアシステム
- 1ライン消去: 100点 × レベル
- 2ライン消去: 300点 × レベル
- 3ライン消去: 500点 × レベル
- 4ライン消去: 800点 × レベル
- コンボボーナス: 1.5倍（連続消去時）

### レベルシステム
- 初期レベル: 1
- レベルアップ条件: 10ライン消去毎
- 落下速度: 1000ms × (0.9 ^ (レベル-1))
- 最高速度: 100ms

## 技術実装

### アーキテクチャ
```
テトリスゲーム
├── GameState（ゲーム状態管理）
├── Tetromino（テトロミノクラス）
├── AudioSystem（音楽・効果音）
├── Renderer（描画システム）
└── TetrisGame（メインゲームロジック）
```

### ゲーム状態管理
```javascript
class GameState {
    - board: 2D配列（ゲームボード）
    - currentPiece: 現在のテトロミノ
    - nextPiece: 次のテトロミノ
    - score: 現在スコア
    - lines: 消去ライン数
    - level: 現在レベル
    - isPlaying: ゲーム中フラグ
    - isPaused: 一時停止フラグ
}
```

### 音楽システム
Web Audio APIを使用してコロベイニキのメロディーを演奏：

```javascript
// 音符データ構造
{
    freq: 周波数（Hz）,
    duration: 長さ（ミリ秒）
}
```

#### コロベイニキ楽譜データ
元の楽譜記法から周波数への変換：
- E四分 → 329.63Hz, 500ms
- B↓八分 → 246.94Hz, 250ms
- 休み四分 → 0Hz, 500ms

### レンダリングシステム
- Canvas 2D APIを使用
- 60fps ゲームループ
- ピクセルパーフェクト描画
- レトロカラーパレット

### データ永続化
- ハイスコア: localStorage
- 設定（音楽ON/OFF）: localStorage
- 形式: JSON

## パフォーマンス

### 最適化項目
- ゲームループ: requestAnimationFrame使用
- 描画: 変更部分のみ再描画
- メモリ: オブジェクトプールによるGC負荷軽減
- 音楽: 低遅延Web Audio API

### 計測値（参考）
- 初期ロード時間: ~200ms
- フレームレート: 60fps（安定）
- メモリ使用量: ~5MB
- CPU使用率: ~2%（アイドル時）

## トラブルシューティング

### よくある問題

#### 音楽が再生されない
- **原因**: ブラウザの自動再生ポリシー
- **解決**: ゲーム開始前にページをクリック

#### 動作が重い
- **原因**: ハードウェアアクセラレーション無効
- **解決**: ブラウザ設定でハードウェアアクセラレーション有効化

#### キー入力が効かない
- **原因**: フォーカスがゲーム画面にない
- **解決**: ゲーム画面をクリックしてフォーカス

## 開発者向け情報

### デバッグ
コンソールログ出力レベル：
- `🎮` : ゲーム状態
- `🧩` : テトロミノ操作
- `🎵` : 音楽関連
- `💯` : スコア関連
- `❌` : エラー

### カスタマイズポイント
1. **色の変更**: `TETROMINOS`オブジェクトの`color`プロパティ
2. **音楽の変更**: `korobeinikiMelody`配列
3. **ゲーム設定**: `GAME_CONFIG`オブジェクト
4. **レトロエフェクト**: CSS `@keyframes`
